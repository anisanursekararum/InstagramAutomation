# InstagramAutomation

app for practice : Instagram

recording video when project running : https://drive.google.com/file/d/1RsM_VHAaDCGzkl70dQY4a1Bby_bVeAc2/view?usp=sharing

Install WebDriverIO

```sh
npm init wdio
npx wdio config
npm install ts-node
```

Run project

```sh
npx wdio wdio.conf.js
```

List Test Cases :

1. Search user
2. Follow user
3. Send messages to user
